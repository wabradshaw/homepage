PET PROJECTS

Palettest

Palettest is a Java library designed to help people test images. It contains a whole suite of functions designed to make every part of testing images easier. Some of these functions are designed to save you writing a few lines of code, but the bit that is unique is the ability to easily test the color palette being used in the image.

Cartographix

A way of drawing stylised maps, like the one in the Location section. Pick a place, pick a colour, and see what it looks like. Just don't pick anything on a small screen because I've not made it a responsive design yet.
